## 如何使用

### 安装依赖

```
yarn install 或 npm install
```

### 启动开发者h5模式

```
npm run serve
```

### vue打包

```
npm run build
```

### 启动开发者electron模式

```
npm run electron:serve
```

### electron打包

```
npm run electron:build
```

### lints校验

```
npm run lint
```

### 自定义vueCli配置

看 [Configuration Reference](https://cli.vuejs.org/config/).
